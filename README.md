这是一只恐龙，路过的人必须夸它可爱。

## 支持：

- 随机/固定浏览
- 随机头图
- 自动获取标题链接
- 中文排版优化
- 行内元素转换
- 块内容分享至某处
- 配置某些引用下的子元素展示（id 截取生成的）创建时间
- 左上边栏扩充

每项的具体说明见 `src/siyuan-*/README.md`
